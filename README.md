Description : Application de gestion académique pour un institut d'enseignement supérieur.
Fonctionnalités : Gestion des niveaux, des étudiants (inscriptions, liste du meilleur étudiant), et des évaluations.
Règle de calcul : La moyenne exclut les TP et ne prend que les devoirs et examens. C'est un point clé du projet
